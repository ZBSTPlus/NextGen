--- layout: grid_layout title: Find the buzz numbers description: | In this game the Player must select all the happy numbers exist in the given grid in the given time limit. rules: | Select all the happy numbers which are displayed. A happy number is a natural number Sum of square of individual digits until sum becomes single digit and final digit sum =1.then it is called happy number Example: n=13 Sum = 1*1+3*3=1+9=10 Sum = 1*1+0*0=1 (single digit) Try to finish the game within the time limit. You should not exceed more than 8 wrong answers ---
