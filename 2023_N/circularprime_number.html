--- layout: grid_layout title: Find the Circular Prime Numbers description: | In this game the Player must select all the circular prime numbers exist in the given grid in the given time limit. rules: | Select all the circular prime numbers which are displayed. A circular prime is a prime number with the property that the number generated at each intermediate step when cyclically permuting its digits will be prime. Try to finish the game within the time limit. You should not exceed more than 8 wrong answers. ---
